La idea principal de esta página es la divulgación de información relacionada a la coctelería. 
Se compone de una sección para consultar recetas de cócteles, una que incluye un formulario de cálculo para facilitar el costeo de una receta, y finalmente una sección de formulario de consultas o contacto.
